# AWM Paper — Public Mirror

This repository hosts the latest compiled PDF of the **Autonomous World Model** paper.

📄 **Read the paper:** [main.pdf](./main.pdf)

## About

The paper's source (LaTeX, drafts, build scripts) lives in a private repository. This public mirror exists solely to share the most recent build of the PDF with a stable, link-friendly URL.

- **Direct view:** https://github.com/thermau5/awm-paper-share/blob/main/main.pdf
- **Raw download:** https://github.com/thermau5/awm-paper-share/raw/main/main.pdf

## Updates

`main.pdf` is automatically synced from the private source repository whenever a new build is pushed. The link above always points to the most recent version — no need to refresh bookmarks.

> If you've viewed the PDF before and want to make sure you're seeing the latest version, do a hard refresh (Ctrl+Shift+R / Cmd+Shift+R) to bypass your browser cache.
